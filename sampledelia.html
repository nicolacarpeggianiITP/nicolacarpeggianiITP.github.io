Sampledelia

A multimedia performance piece that uses images and sounds from everyday life moments, to recreate an immersive environment. This environment will be based on sounds and images, happened in a precise time and location, that are sampled from their original sources and mixed together in a musical “non-time” recreated in an immersive room. This system was created using MAX MSP JITTER, a multichannel audio system and multiple projectors mapped using VDMX.
